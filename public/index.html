Update(5/12/2015): There's been an update (check the reddit post for more details) that now uses a different format for cookies and state. You'll need to re-enter in your weapons (sorry!) but you can save the artifact portion of the save state (everything before the first "|"). You'll want to copy that portion, reset all cookies, refresh (everything will be defaults), then replace the default artifact portion with yours, then load from save state, and re-enter your weapons.
